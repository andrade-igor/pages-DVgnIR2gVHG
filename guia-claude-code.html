Como iniciar no Claude Code

Instale e configure o Claude Code em 15 minutos — mesmo sem saber programar. Sem enrolação. Passo a passo direto ao ponto.

Salve este guia no seu celular Baixe o arquivo e consulte offline, quando quiser.
Tempo estimado 15 minutos

O que você vai conseguir

Esse guia não é teoria. É o caminho mais curto entre você e a ferramenta de IA mais poderosa disponível hoje para quem quer resolver problemas reais direto no seu editor de código.

Não precisa saber programar. Não precisa de experiência com terminal. Vamos usar o VS Code — um editor gratuito e visual. Só precisa seguir os passos.

Resultado ao final

Você terá o Claude Code funcionando dentro do VS Code no seu computador, pronto para automatizar tarefas, gerar código e resolver problemas com IA — tudo em uma interface visual e amigável.

1 Passo 1 de 7

Entenda o que é o Claude Code

Claude Code não é um chatbot comum. É uma IA que roda direto dentro do VS Code — o editor de código mais usado do mundo. Você digita o que precisa em português e ela executa.

Isso importa porque o Claude Code tem acesso direto aos seus arquivos, projetos e sistemas. Sem copiar e colar entre abas. Ele lê, edita e cria arquivos sozinho, dentro do seu ambiente de trabalho.

Claude.ai (Chat Web)

Conversa em texto. Você copia respostas e cola no seu projeto. Limitado ao que cabe na janela de chat.

Claude Code (VS Code)

Acessa seus arquivos diretamente. Lê, edita, cria e executa. Trabalha dentro do seu projeto real, com interface visual.

Dica

Pense no Claude.ai como um consultor que te responde por mensagem. O Claude Code é esse mesmo consultor sentado na sua mesa, com acesso ao seu computador — e você vê tudo acontecendo na tela.

Resultado deste passo

Você entende por que profissionais estão migrando para essa ferramenta.

2 Passo 2 de 7

Crie sua conta na Anthropic

O Claude Code precisa de uma conta com plano pago. O plano gratuito do Claude.ai não inclui acesso ao Claude Code.

Como criar

  1. Acesse claude.ai/login
  2. Crie sua conta com Google ou email
  3. Escolha um plano que inclua Claude Code

Qual plano escolher

Pro
$20/mês
Acesso completo ao Claude Code. Comece por aqui — é o suficiente para sentir o poder da ferramenta e decidir se faz sentido pra você.
Max
$100–200/mês
Modelo mais capaz e contexto estendido. Migre quando sentir que o Pro já não dá conta da sua demanda.
Minha recomendação

Começa pelo Pro. Sente o gosto, entende o poder. Se fizer sentido pro teu dia a dia (e vai fazer), aí tu migra pro Max quando quiser. Eu uso o Max de $200/mês e vale cada centavo — mas eu só sei disso porque comecei pelo Pro.

Resultado deste passo

Conta ativa com plano que inclui Claude Code.

3 Passo 3 de 7

Instale o VS Code e a extensão Claude Code

Vamos usar o VS Code (Visual Studio Code) — um editor gratuito da Microsoft. É a forma mais simples de usar o Claude Code, com interface visual e sem precisar decorar comandos de terminal.

Passo 3.1: Baixe o VS Code

  1. Acesse code.visualstudio.com
  2. Clique no botão de download (ele detecta seu sistema automaticamente)
  3. Instale normalmente — pode manter todas as opções padrão
Dica

Na instalação do Windows, marque a opção "Add to PATH" se aparecer. E marque "Open with Code" para poder abrir pastas com clique direito.

Passo 3.2: Instale a extensão Claude Code

  1. Abra o VS Code
  2. Na barra lateral esquerda, clique no ícone de Extensões (parece 4 quadradinhos) ou pressione Ctrl+Shift+X
  3. Na busca, digite Claude Code
  4. Encontre a extensão oficial da Anthropic e clique em Install

Após instalar, você vai ver o ícone do Claude Code na barra lateral esquerda do VS Code.

Atenção

Certifique-se de instalar a extensão oficial da Anthropic. Existem imitações — confira o nome do publicador antes de instalar.

Resultado deste passo

VS Code instalado com a extensão Claude Code pronta para usar.

4 Passo 4 de 7

Faça login e conecte sua conta

Agora você vai conectar o Claude Code à conta que criou no Passo 2. Só precisa fazer isso uma vez.

Passo a passo

  1. No VS Code, clique no ícone do Claude Code na barra lateral esquerda
  2. Clique em "Sign In" (ou "Entrar")
  3. O navegador vai abrir automaticamente pedindo autenticação
  4. Faça login com a mesma conta do Passo 2
  5. Volte ao VS Code — o painel do Claude Code vai aparecer conectado

Suas credenciais ficam salvas. Na próxima vez que abrir o VS Code, já entra conectado automaticamente.

Resultado deste passo

Claude Code autenticado e pronto para usar dentro do VS Code.

5 Passo 5 de 7

Seu primeiro comando

Hora de ver a IA agindo no seu computador. Vamos fazer dois testes simples.

Abra uma pasta no VS Code

No VS Code, vá em File > Open Folder (Arquivo > Abrir Pasta) e escolha qualquer pasta do seu computador. Pode ser a Área de Trabalho ou uma pasta de projeto.

Depois, clique no ícone do Claude Code na barra lateral para abrir o painel de chat.

Teste 1: Pergunte algo

No painel do Claude Code, digite em português:

o que tem nessa pasta?Claude Code

O Claude vai analisar todos os arquivos da pasta e te dar um resumo. Ele lê o conteúdo real, não só os nomes.

Teste 2: Crie algo

Agora peça para ele criar algo:

crie um arquivo HTML com uma página pessoal bonita sobre mimClaude Code

O Claude vai criar o arquivo na pasta, escrever o código, e te mostrar o que fez. Você vê o arquivo aparecendo em tempo real no VS Code. Você só precisou pedir em português.

Comandos úteis

No painel do Claude Code, você pode usar:

  • /help — lista todos os comandos disponíveis
  • /cost — mostra quanto você gastou na sessão
  • /clear — limpa a conversa atual
Resultado deste passo

Você deu seu primeiro comando e viu a IA agir no seu computador.

Comandos essenciais do dia a dia

Esses são os recursos que você vai usar constantemente dentro do VS Code. Não precisa decorar tudo agora — salve esse guia e consulte quando precisar.

Ação O que faz

7 Passo 7 de 7

Como conversar com o Claude (mini guia de prompt)

Essa é a parte mais importante do guia. A ferramenta é poderosa, mas o resultado depende de como você pede.

Regra de ouro

Trate o Claude como se fosse uma criança muito inteligente. Ele é capaz de fazer coisas incríveis — mas precisa que você explique com clareza o que quer, o que não quer, e o contexto da situação. Se você der uma instrução vaga, o resultado vai ser vago. Se der uma instrução detalhada, o resultado vai ser exatamente o que você precisa.

A anatomia de um bom pedido

  1. Contexto — Explique a situação. "Estou organizando uma viagem pro Japão com minha esposa..."
  2. O que você quer — Seja direto. "Preciso de um roteiro de 10 dias com foco em comida e cultura..."
  3. O que você NÃO quer — Elimine caminhos errados. "Não quero pontos turísticos óbvios, não quero roteiro corrido..."
  4. Resultado esperado — Descreva o formato. "Me entrega dia a dia, com nome dos lugares e links."

Na prática: antes e depois

Instrução vaga

"me ajuda com emails"

Instrução com contexto

"Estou organizando uma viagem pro Japão com minha esposa, 10 dias, em outubro. Quero um roteiro focado em comida de rua e cultura local. Nada de ponto turístico óbvio. Me entrega dia a dia, com nome dos lugares, bairro, e o que fazer em cada um."

Percebe a diferença? O segundo pedido não deixa espaço pra interpretação errada. Você contextualizou a situação, disse o que quer, o que não quer, e como quer receber.

Mais exemplos práticos

Vago

"faz uma landing page"

Detalhado

"Cria uma página pessoal pra mim. Sou designer freelancer. Tom criativo e descontraído. Seções: uma frase de impacto sobre mim, meus 3 melhores trabalhos com imagem, um formulário de contato. Cores escuras com detalhes em roxo."

Vago

"analisa meu projeto"

Detalhado

"Analisa todos os arquivos desse projeto e me diz: qual a estrutura de pastas, quais tecnologias estão sendo usadas, e se tem algum problema óbvio de organização. Me responde em formato de lista."

Dicas rápidas

  • Quebre tarefas grandes em etapas — em vez de pedir tudo de uma vez, vá passo a passo
  • Abra a pasta certa no VS Code — sempre use File > Open Folder na pasta do projeto que você quer trabalhar
  • Explore antes de pedir mudanças — peça pro Claude analisar o que existe antes de alterar
  • Se o resultado não ficou bom, refine — diga o que ficou errado e peça pra ajustar. É uma conversa, não um comando único
Atenção

O Claude Code pode criar, editar e deletar arquivos. Ele sempre pede permissão antes, mas leia o que ele propõe antes de aprovar.

Você sabe se comunicar com o Claude de forma que ele entrega exatamente o que você precisa.
